总结这学期

发表于 2016-12-14

撒花，688和601都结课了，上学期有cc，这学期有688和601，一直没去上课最后一节课见到了Zico, 哈哈。没想到Zico还是蛮帅的，一学期学到的东西还挺多，这里把学到的技能都盘点一下:

###15688 Practical Datascience:
Regression, Classification, NLP, TFIDF, Hypothesis test, Decision tree and boosting, K-means, PCA, Probabilistic modeling, Map Reduce, Recommendation System, Deep learning, MLE, Stastics model, Python Scraper, Pandas, Numpy, Scikit learn, sqlite, Pagerank, Djisktra’s algorithm,Gaussian Mixture Model. 很多经典算法都用python实现了一遍加深理解。

Assignment 1，自己写了个parser，基本各种玩正则，优化了一下，可以抓到a href的各种tag，比beautiful soup好用，哈哈。

Assignment 2，processing bus stop data , Time series 算sliding average。自己写一个graph类，实现Djisktra’s algorithm 和 pagerank.

Assignment 3, Text processing, scikit learn lemmatizer process features, NLP 手写tfidf, cosine_similarity, n-gram, perplexity.

Assignment 4, Cross validation, 实现NB (Guassian 和 Categorical), Analyze mnist 算softmax loss 和 gradient descent

Assignment 5, Unsupervised learning, 实现Kmeans, 算Entropy,实现GM算法，最后去predict faces，直接用svm就可以达到很高正确率了，据说最好的人用的是
a simple single layer textbook neural network with 100 hidden units and sigmoid activation functions, and used that for classification on the raw pixel data. 90%+的正确率，厉害了word 哥。Collaborative filtering 用了经典的movie data，用了Matrix Factorization approach，推荐给不同user. Hypothesis test用了weighted的data预测总统选举。

Assignment 6, Deep learning, 这个谈的比较范范，重在实践所以用了Keras library, 不得不说比601自己写cnn爽多了，Map reduce, 这个上学期cc不知道做了多少次，也就是几个比较经典的应用了, linked network, k-means, word count. 最后一个作业用pymc library 去predict是哪一种用电，就是各种花式玩library。

Tutorial 我做的XGBOOST, 真的是没时间研究都比较水。。final project做的airbnb review 和user behavior，都比较水啦。。。

总的来说688还是比较实用的一门课，至少上完了课python和numpy,pandas的使用还是比较熟练了，各种library也可以用起来了，TA非常给力啊，Eric基本7x24小时piazza回答问题，啥都不说了，就是感动。

10601 Machine Learning

这门课一开始选了pf所以也没好好学，作业还是好好做的，作为ml的入门课，不小心选到了section B,结果变成了一门纯理论stats的课..TUT从此打消了我搞data science的念头。。实在是头发要掉光，谁叫我数学不好呢。。B走的..是传统路线，用的都是matlab和Octave(浓浓的数学风), 卧槽0 0之前从来没用过的我。。这门课据说之前都是Eric Xing和Tom Mitchel(大师啊)教的，后面就变成了。。唉。。
什么都不说了，先来一把提纲:
Decision Tree, The Naive Bayes algorithm , Logistic Rgression, Linear Regression,Perceptrons and Large Margin ,Kernels, SVM, Generalization and Overfitting: Sample Complexity Results ,Model Selection , Clustering,Neural Networks ,Deep Learning, Boosting, Active Learning , Semi-Supervised Learning ,Graphical Models, HMMs and CRFs (这个厉害了反正我是弃疗了),Expectation Maximization,Reinforcement Learning ,Matrix Factorization and Collaborative Filtering ..
作业基本就是几大张纸的数学题，autolab上的matlab题，反正什么NB, LR, SVM, CNN都实现了一遍，感觉很酸爽啊，虽然收获挺多（至少入了个门），课上厉害的人很多，反正我一个pf就没什么好说的了，相比较而言，data这种东西，还真是学数学的比较好啊。。。我还是适合码码代码。。

J2EE Web Service

比较水，虽然是大牛Jeff Eppergine教的。。最后老师还给我一张彩票。。到现在都没看我有没有中奖。。就把web service讲了一遍，什么transaction那，servlet啦，都是比较简单的。。

Distributed System

老师讲的虽然烂，但是用心学还是能学很多，比如各种network protocol啦，distributed file system，jms，web service, RPC, RMI, time algorithm, distributed transactions.为了考试我还是默默地学习了一下， 可能之后会出个总结帖，但是以后搞infrastructure必须学的好才行啊。。。
