Permutation

不sorted的版本很简单，sorted的版本关键在于
if (visited[i] == 1 ||
(i > 0 && nums[i] == nums[i-1] && visited[i-1] == 0)) {

Sliding Window

424.Longest Repeating Character Replacement
简直被难倒了，这道题还以为用dp，结果sliding window，但是思想需要掌握，一般有k的数列题，不是特别明显用priorityqueue和dp的那种，基本就是sliding window了！！
思想就是：用一个window维护一个窗口，这个窗口里记录最长的字符串max，
1.max+k大于end-start，不管，window向后延伸就好，然后每次都更替一下现在的最大长度（end－start不一样噢，max不一样了）。
2.max＋k小于end－start的时候，更新start＋＋，让窗口向后移！
